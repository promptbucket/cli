# PromptBucket CLI (`pbt`)

Package and manage prompts using simple YAML manifests.

## Commands

- `pbt init` – scaffold `promptbucket.yaml`.
- `pbt build` – create a `.pbt` archive.
- `pbt completion` – generate shell completions.

## Building

```bash
make vet test build
```
